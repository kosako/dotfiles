## Decisions

- repository 名は `dotfiles` にする。
- 初期は物理 repo を分けず、`dotfiles` ひとつで始める。
- 将来必要になったら overlay repo 分割を検討する。
- `profile` は権限そのものではなく、`capabilities` のプリセットとして扱う。
- 開発 project は `~/src/...` 階層に置く。
- `npmHardeningMode` は `off | report | enforce` にする。
- `corepackMode` は `off | report | enable` にする。
- Git signing は後続 `git-signing` module として扱う。
- VS Code は settings のみ管理し、extensions は自動 install しない。
- 古い `/Users/kosako/dev/portable-mac-dev-env` は削除した。
